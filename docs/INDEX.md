## üöÄ Quick Start

### For Users:
1. **Add Bot to Group:** Simply add @NexusBot to any Telegram group
2. **Use Commands:** Type `/help` to see all commands
3. **Check Documentation:** Read [COMPLETE_COMMANDS_REFERENCE.md](../COMPLETE_COMMANDS_REFERENCE.md) for command details
4. **Open Mini App:** Click the menu button to access the admin panel

### For Developers:
1. **Clone Repository:** `git clone <repository>`
2. **Configure Environment:** `cp .env.example .env && nano .env`
3. **Start Services:** `docker-compose up -d`
4. **Check Logs:** `docker-compose logs -f bot`
5. **Read Documentation:** All docs are in `/docs/` folder

### For Deployment:
1. **Render (Recommended):** Use `render.yaml` blueprint
2. **Docker on VPS:** Use `docker-compose.yml`
3. **Local Development:** Use `docker-compose.yml` for local testing
